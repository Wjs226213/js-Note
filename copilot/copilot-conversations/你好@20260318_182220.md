## 基本语法

### 关键字
- 特殊含义的英文字符

### 常量
程序执行过程中值不会发生改变的数据（数值，不是计算过程）。

**类型：**
  - 整型常量
  - 实型常量
  - 字符常量：单引号括起来的字母、数字、英文符号（只能有一个）
  - 字符串常量：双引号括起来的字符串

**输出格式说明符：**
| 数据类型 | 格式符 |
|----------|--------|
| 整型     | %d     |
| 实数     | %f     |
| 字符串   | %s     |
| 字符     | %c     |

### 变量
可以发生改变的数据。变量不可以重复定义，一条语句可以定义多个变量。

**使用条件：** 先赋值，再使用

### 进制

**前缀表示：**
  - 二进制：`0b` 开头
  - 八进制：`0` 开头
  - 十六进制：`0x` 开头

**进制转换方法：**
  - **任意进制转十进制：** 系数 × 基数的权次幂，然后相加
    - 系数：当前位置的数值
    - 基数：进制数
    - 权次幂：从右往左从0开始计数
  - **十进制转其他进制：** 除基取余法（不断除以目标进制基数，取余数）

## 高级运算符

### 自增/自减运算符
| 操作     | 符号 | 说明         |
|----------|------|--------------|
| 先自增   | ++a  | 先加后用     |
| 先自减   | --a  | 先减后用     |
| 后自增   | a++  | 先用后加     |
| 后自减   | a--  | 先用后减     |

**注意：** 不同系统的运算顺序可能不同：
- **Windows机制：** 前缀运算优先于后缀，统一完成自增/自减后再计算
- **Mac/Linux机制：** 每个前缀和后缀都是独立个体

**示例（Windows机制）：**
```c
#include <stdio.h>
int main() {
    int a = 10;
    int k1 = a++ + a++ + a;   // 10+10+10=30, a=12
    int k2 = ++a + ++a + a;   // 14+14+14=42, a=14
    int k3 = ++a + a++ + a;   // 15+15+15=45, a=16
    int k4 = a++ + ++a + a;   // 17+17+17=51, a=18
}
```

### 关系运算符
| 运算符 | 含义     |
|--------|----------|
| ==     | 等于     |
| !=     | 不等于   |
| >      | 大于     |
| >=     | 大于等于 |
| <      | 小于     |
| <=     | 小于等于 |

**返回值：** 0（false）或 1（true）

### 逻辑运算符
| 运算符 | 含义                     | 结果条件         |
|--------|--------------------------|------------------|
| &&     | 逻辑与（同时满足）       | 全部为真         |
| \|\|   | 逻辑或（一个满足）       | 至少一个为真     |
| !      | 逻辑非（取反）           | 真变假，假变真   |

**应用示例：**
```c
#include <stdio.h>
int main() {
    // 判断数字不含7且不是7的倍数
    int a;
    scanf("%d", &a);
    int ge = a % 10;
    int shi = a / 10 % 10;
    printf("%d", (ge != 7 && shi != 7 && a % 7 != 0));
}
```

#### 短路效应
- **&&（逻辑与）：** 左边为假时，右边不执行
- **\|\|（逻辑或）：** 左边为真时，右边不执行

```c
#include <stdio.h>
int main() {
    int a = 1, b = 5;
    a > 0 && ++b;       // b自增，输出：a=1, b=6
    a > 0 || ++b;       // b不自增，输出：a=1, b=5
}
```

### 三元运算符
**比较两个数的最大值：**
```c
#include <stdio.h>
int main() {
    int a = 10;
    int b = 20;
    int c = a > b ? a : b;
    printf("max = %d", c);
}
```

**比较三个数的最大值：**
```c
#include <stdio.h>
int main() {
    int a = 10, b = 20, c = 30;
    int temp = a > b ? a : b;
    int final_max = temp > c ? temp : c;
    printf("max = %d", final_max);
}
```

### 逗号运算符
> 特性：
>   - 从左往右执行
>   - 优先级最低
>   - 最后一个子表达式的结果作为整个表达式的结果

```c
#include <stdio.h>
int main() {
    int i = 0;
    printf("%d", (i = 3, ++i, i++, i + 5));
}
```

### 运算符优先级
> 重要规则：
>   1. 小括号 `()` 优先级最高
>   2. 一元运算符 > 二元运算符 > 三元运算符
>   3. `&&` > `\|\|` > 赋值运算符
>   4. **注意：** `>=` 的优先级高于 `!=`

**优先级练习：**
```c
#include <stdio.h>
int main() {
    // 示例1：三元运算符嵌套
    int w = 4, x = 3, y = 2, z = 1;
    int number = w < x ? w : (z < y ? z : x);
    printf("number = %d", number);
    
    // 示例2：逻辑运算符优先级
    int a = 3, b = 4, c = 5;
    int num = a || (b + c && b - c);
    printf("num = %d", num);
    
    // 示例3：综合优先级
    int num2 = !(((a < b) && !c) || 1);
    printf("num2 = %d", num2);
}
```

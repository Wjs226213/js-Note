## 一、大数据的基本概念与特征

### 1. 定义

- **大数据（Big Data）**：指无法在一定时间范围内用常规软件工具进行捕捉、管理和处理的数据集合
- 需要新处理模式才能具有更强的决策力、洞察力和流程优化能力

### 2. 5V 特征

| 特征 | 英文 | 含义 |
|------|------|------|
| **Volume（大量）** | 数据量巨大 | 从TB级到PB、EB级 |
| **Velocity（高速）** | 数据产生和处理速度快 | 实时或近实时处理 |
| **Variety（多样）** | 数据类型多样 | 结构化、半结构化、非结构化 |
| **Value（价值）** | 价值密度低 | 需从海量数据中挖掘价值 |
| **Veracity（真实性）** | 数据质量 | 数据的准确性和可信度 |

> ⚠️ **核心考点**：大数据的 5V 特征必须记住

### 3. 数据类型

| 类型 | 特点 | 举例 |
|------|------|------|
| **结构化数据** | 有固定模式 | 关系数据库表 |
| **半结构化数据** | 有标签但无固定模式 | JSON、XML、HTML |
| **非结构化数据** | 无固定结构 | 文本、图片、视频、音频 |

## 二、大数据采集与处理框架

### 1. 数据采集

- **日志采集**：Flume、Logstash、Scribe
- **网络爬虫**：Scrapy、Heritrix
- **传感器采集**：IoT设备
- **数据库同步**：CDC（Change Data Capture）

### 2. 大数据处理框架

```
数据采集 → 数据存储 → 数据处理 → 数据分析 → 数据可视化
```

| 层次 | 工具/框架 |
|------|----------|
| **采集层** | Flume、Kafka、Sqoop |
| **存储层** | HDFS、HBase、MongoDB |
| **计算层** | MapReduce、Spark、Flink |
| **分析层** | Hive、Pig、Mahout |
| **可视化层** | ECharts、Tableau |

### 3. Hadoop 生态系统

- **HDFS**：分布式文件系统（存储层）
- **YARN**：资源调度框架
- **MapReduce**：分布式计算框架
- **Hive**：数据仓库（SQL查询）
- **HBase**：分布式列式数据库
- **ZooKeeper**：分布式协调服务

## 三、大数据关键技术

### 1. MapReduce

- **核心思想**：**分而治之**
- **两个阶段**：

```
Map（映射）阶段 → Shuffle（洗牌）阶段 → Reduce（归约）阶段
```

- **Map**：将输入数据分解为键值对 (key, value)
- **Shuffle**：按 key 分组排序
- **Reduce**：对相同 key 的数据进行聚合
- **WordCount 示例**：
  - Map: `"hello world"` → `(hello, 1), (world, 1)`
  - Reduce: `(hello, [1,1,1])` → `(hello, 3)`

### 2. 分布式数据挖掘

- **特点**：数据分散在多个节点上，需要分布式算法
- **关键技术**：
  - 数据并行：将数据分片，各节点并行处理
  - 模型并行：将模型分片到不同节点
  - 参数服务器：分布式参数同步
- **挑战**：通信开销、负载均衡、容错性

### 3. Spark 机器学习（MLlib）

- **Spark 优势**：基于内存计算，比 MapReduce 快 10-100 倍
- **MLlib 提供**：
  - 分类：逻辑回归、决策树、随机森林
  - 回归：线性回归、岭回归
  - 聚类：K-Means、LDA
  - 协同过滤：ALS（交替最小二乘法）
  - 特征工程：TF-IDF、Word2Vec、PCA

### 4. 大数据存储

| 存储方案 | 特点 | 适用场景 |
|---------|------|---------|
| **HDFS** | 分布式文件系统 | 大文件批量存储 |
| **HBase** | 列式存储 | 随机读写、实时查询 |
| **MongoDB** | 文档型数据库 | JSON数据存储 |
| **Redis** | 内存数据库 | 缓存、实时计算 |
| **Elasticsearch** | 搜索引擎 | 全文检索 |

## 四、大数据应用价值

### 1. 日常生活应用

| 领域 | 应用 |
|------|------|
| **电商** | 个性化推荐（淘宝、京东） |
| **社交** | 好友推荐、内容推荐（微信、抖音） |

| **教育** | 个性化学习、学情分析 |

### 2. 产业应用

- **智能制造**：生产优化、质量检测
- **智慧城市**：交通管理、城市规划
- **精准农业**：产量预测、病虫害检测
- **能源**：智能电网、能耗优化

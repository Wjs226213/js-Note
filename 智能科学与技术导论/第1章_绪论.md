# 第1章 绪论

> 考试题型：单选题、判断题、填空题、简答题、综合题

## 考核要点

1. 了解人类智能与机器智能之间的关系
2. 了解智能科学与技术的概念、内容、研究内容、发展历史

---

## 一、人类智能与机器智能的关系

### 1. 人类智能（Human Intelligence）

- 人类智能是指人类在认识、改造世界过程中，运用**思维**和**知识**解决问题的能力
- 核心要素：感知、认知、学习、推理、创造、情感等
- 人类智能具有**自适应性**、**创造性**、**情感性**、**社会性**

### 2. 机器智能（Machine Intelligence）

- 机器智能是指机器模拟、延伸和扩展人类智能的能力
- 通过**算法**、**数据**和**计算**来实现
- 典型形式：专家系统、机器学习、深度学习、自然语言处理等

### 3. 两者关系

| 比较维度 | 人类智能     | 机器智能      |
| ---- | -------- | --------- |
| 基础   | 大脑神经元    | 算法+数据+算力  |
| 学习方式 | 少样本、迁移学习 | 大量数据训练    |
| 创造力  | 具备原创性思维  | 基于已有模式组合  |
| 情感   | 有情感体验    | 可模拟但无真正情感 |
| 计算速度 | 相对较慢     | 极快        |
| 持久性  | 易疲劳      | 不知疲倦      |
| 泛化能力 | 强        | 有限        |

> **关键理解**：机器智能是对人类智能的**模拟、延伸和扩展**，而非**替代**。两者是互补关系。

---

## 二、智能科学与技术的基本概念

### 1. 定义

- **智能科学与技术**：研究如何用机器模拟、延伸和扩展人类智能的理论、方法和技术的学科
- 是一门**交叉学科**，融合了计算机科学、认知科学、控制论、信息论等多个领域

### 2. 研究内容

- **基础理论**：认知模型、知识表示、推理机制、学习算法
- **关键技术**：机器感知、模式识别、机器学习、自然语言处理
- **应用系统**：智能机器人、智能交通、智能医疗、智能制造等

### 3. 学科特点

- **交叉性**：涉及计算机、认知科学、数学、控制科学等
- **实践性**：理论与应用紧密结合
- **前沿性**：处于科技发展最前沿

## 三、发展历史

| 阶段        | 时间          | 主要事件                              |
| --------- | ----------- | --------------------------------- |
| **萌芽期**   | 1940s-1950s | 图灵提出"机器能思考吗"；McCulloch-Pitts神经元模型 |
| **诞生期**   | 1956年       | **达特茅斯会议**，"人工智能"正式诞生             |
| **黄金期**   | 1956-1974   | 感知机、专家系统、LISP语言                   |
| **第一次低谷** | 1974-1980   | 计算能力不足、感知机局限性暴露                   |
| **复兴期**   | 1980-1987   | 专家系统商业化、日本第五代计算机                  |
| **第二次低谷** | 1987-1993   | 专家系统瓶颈、投资撤退                       |
| **稳步发展**  | 1993-2010   | 统计学习兴起、SVM、贝叶斯方法                  |
| **爆发期**   | 2011-至今     | 深度学习突破、AlphaGo、大语言模型              |

> ⚠️ **重要考点**：1956年达特茅斯会议是人工智能诞生的标志性事件

## 常见考点总结

### 可能出的单选题

1. 人工智能诞生的标志性事件是？ → **1956年达特茅斯会议**
2. 图灵在哪一年提出了著名的"图灵测试"？ → **1950年**
3. 智能科学与技术是一门什么学科？ → **交叉学科**

### 可能出的判断题

1. 机器智能可以完全替代人类智能 → **×**（是模拟、延伸和扩展）
2. 1956年达特茅斯会议标志着人工智能的诞生 → **√**

### 可能出的填空题

1. 智能科学与技术是研究如何用机器____、____和____人类智能的学科 → 模拟、延伸、扩展
2. 人工智能诞生于____年的____会议 → 1956、达特茅斯

### 可能出的简答题

1. 简述人类智能与机器智能的关系
2. 简述智能科学与技术的研究内容

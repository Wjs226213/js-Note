# 第3章 机器感知及其应用

> 参考教材：《智能科学与技术导论》朱珍 机械工业出版社 2021年第1版

1. 了解机器感知的物理原理、特性与要求

4. 理解基于听觉感知的声源定位与跟踪的基本原理

## 一、机器感知概述

### 1. 定义

- **机器感知**：使机器能够像人类一样通过传感器获取和理解外部环境信息的技术
- 本质是用机器**模拟人类的感觉器官**（眼、耳、鼻、皮肤等）

### 2. 物理原理

| 感知类型 | 物理原理 | 传感器 |
|---------|---------|--------|
| **视觉感知** | 光学成像 | 摄像头、深度相机 |
| **听觉感知** | 声波传播 | 麦克风阵列 |
| **触觉感知** | 力/压力感应 | 力传感器、触觉传感器 |
| **嗅觉感知** | 气体化学反应 | 电子鼻 |
| **温度感知** | 热辐射 | 红外传感器 |

### 3. 特性与要求

- **实时性**：快速响应环境变化
- **准确性**：精确获取目标信息
- **鲁棒性**：在噪声、遮挡等复杂环境下仍能工作
- **多尺度**：能感知不同大小、距离的目标

---

## 二、多模态机器感知

### 1. 概念

- **多模态感知**：同时利用多种传感器（视觉、听觉、触觉等）获取信息
- 融合不同模态的数据以获得**更全面、更准确**的环境理解

### 2. 多模态融合方式

| 融合层次 | 说明 | 特点 |
|---------|------|------|
| **数据层融合** | 原始传感器数据直接融合 | 信息保留完整，但数据量大 |
| **特征层融合** | 提取各模态特征后融合 | 平衡信息量和计算复杂度 |
| **决策层融合** | 各模态分别决策后融合 | 灵活性高，但可能丢失信息 |

### 3. 典型应用

- **视听融合**：唇语识别（视觉+语音）
- **触觉+视觉**：机器人抓取
- **多摄像头融合**：全景监控

## 三、基于视觉感知的目标检测、识别与跟踪

### 1. 目标检测

- **任务**：在图像中找到目标的位置（通常用边界框标注）
- **基本原理**：
  1. 图像预处理（灰度化、去噪、增强）
  2. 特征提取（颜色、纹理、形状、深度学习特征）
  3. 候选区域生成
  4. 分类判断
- **经典方法**：
  - 传统方法：Haar特征 + AdaBoost、HOG + SVM
  - 深度学习方法：**R-CNN系列**、**YOLO**、**SSD**

### 2. 目标识别（分类）

- **任务**：确定检测到的目标属于哪个类别
- **基本流程**：图像输入 → 特征提取 → 分类器 → 类别输出
- **关键网络**：AlexNet、VGG、GoogLeNet、ResNet

### 3. 目标跟踪

- **任务**：在视频序列中持续定位运动目标
- **基本方法**：
  - **生成式跟踪**：建模目标外观（如粒子滤波、卡尔曼滤波）
  - **判别式跟踪**：区分目标与背景（如相关滤波、深度学习）
- **关键步骤**：
  1. 目标初始化（第一帧标注）
  2. 运动估计
  3. 外观模型更新
  4. 位置预测

## 四、基于听觉感知的声源定位与跟踪

### 1. 基本原理

- 利用**麦克风阵列**采集声音信号
- 通过分析不同麦克风之间声音到达的**时间差（TDOA）**或**相位差**来确定声源方向

### 2. 关键技术

| 技术 | 说明 |
|------|------|
| **时间差估计（TDOA）** | 计算声音到达不同麦克风的时间差 |
| **广义互相关（GCC）** | 常用的TDOA估计方法 |
| **波束成形（Beamforming）** | 通过加权求和增强特定方向的信号 |
| **声源跟踪** | 连续帧跟踪声源位置变化 |

### 3. 应用场景

- 智能音箱（如小爱同学、天猫精确定位说话人）
- 视频会议（自动对焦说话人）
- 机器人听觉（定位声源方向）
- 智能监控（声音事件定位）

## 常见考点总结

### 可能出的单选题

1. 广义互相关（GCC）方法主要用于？ → **时间差估计（TDOA）**
2. 多模态融合不包括以下哪个层次？ → 信号层融合（答案应为数据层、特征层、决策层之一）
3. YOLO 属于哪种目标检测方法？ → **基于深度学习的一阶段检测方法**

### 可能出的判断题

1. 多模态感知只包括视觉和听觉 → **×**（还包括触觉、嗅觉等）
2. 声源定位的基本原理是利用不同麦克风的时间差 → **√**

### 可能出的填空题

1. 机器感知主要分为____、____、____等类型 → 视觉感知、听觉感知、触觉感知
2. 多模态融合的三个层次：____、____、____ → 数据层融合、特征层融合、决策层融合

### 可能出的简答题

1. 简述目标检测、识别、跟踪的区别与联系
2. 简述基于麦克风阵列的声源定位原理

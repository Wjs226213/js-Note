## 一、机器学习的基本概念与工作流程

### 1. 基本定义

- **机器学习（Machine Learning）**：计算机系统通过**经验（数据）**自动改进性能的过程
- Tom Mitchell 定义：一个计算机程序从经验 E 中学习任务 T，以性能度量 P 衡量，如果在 T 上的性能 P 随着 E 的增加而提高，则称该程序在学习

### 2. 典型工作流程

```
数据收集 → 数据预处理 → 特征工程 → 模型选择 → 模型训练 → 模型评估 → 模型部署
```

| 步骤 | 内容 |
|------|------|
| **数据收集** | 获取原始数据（爬虫、传感器、数据库等） |
| **数据预处理** | 数据清洗、缺失值处理、归一化、数据增强 |
| **特征工程** | 特征提取、特征选择、特征变换（降维等） |
| **模型选择** | 根据任务类型选择合适的算法 |
| **模型训练** | 用训练集拟合模型参数 |
| **模型评估** | 用测试集评估模型性能 |
| **模型部署** | 将模型部署到生产环境 |

### 3. 关键概念

- **训练集/验证集/测试集**：通常比例为 7:1.5:1.5 或 6:2:2
- **过拟合**：模型在训练集上表现好，测试集上差
- **欠拟合**：模型过于简单，无法捕获数据规律
- **偏差-方差权衡**：偏差高→欠拟合；方差高→过拟合

## 二、机器学习模型的分类

### 1. 按学习方式分类

| 类型 | 特点 | 代表算法 |
|------|------|---------|
| **监督学习** | 有标签数据训练 | 线性回归、决策树、SVM、神经网络 |
| **无监督学习** | 无标签数据 | K-Means、PCA、自编码器 |
| **半监督学习** | 少量标签+大量无标签 | 标签传播、自训练 |
| **强化学习** | 通过与环境交互获得奖励 | Q-Learning、DQN、PPO |

### 2. 按任务类型分类

| 任务类型 | 输出 | 评价指标 |
|---------|------|---------|
| **分类** | 离散类别 | 准确率、精确率、召回率、F1 |
| **回归** | 连续值 | MSE、MAE、R² |
| **聚类** | 簇标签 | 轮廓系数、CH指数 |
| **降维** | 低维表示 | 保留方差比 |

### 3. 常见算法对比

| 算法 | 类型 | 优点 | 缺点 |
|------|------|------|------|
| 线性回归 | 监督/回归 | 简单、可解释 | 只能拟合线性关系 |
| 决策树 | 监督/分类回归 | 直观、无需特征缩放 | 容易过拟合 |
| 随机森林 | 监督/分类回归 | 精度高、不易过拟合 | 计算量大 |
| SVM | 监督/分类 | 小样本效果好 | 大数据集慢 |
| KNN | 监督/分类 | 简单 | 计算量大、受维度影响 |
| K-Means | 无监督/聚类 | 快速、简单 | 需指定K、对初值敏感 |

## 三、人工神经网络的基本原理

### 1. 生物神经元 → 人工神经元

- 生物神经元：树突（输入）→ 细胞体（处理）→ 轴突（输出）
- 人工神经元模型（M-P模型）：

$$y = f\left(\sum_{i=1}^{n} w_i x_i + b\right)$$

- $x_i$：输入
- $w_i$：权重
- $b$：偏置
- $f$：激活函数

### 2. 常用激活函数

| 函数 | 公式 | 特点 |
|------|------|------|
| **Sigmoid** | $f(x) = \frac{1}{1+e^{-x}}$ | 输出(0,1)，易梯度消失 |
| **Tanh** | $f(x) = \frac{e^x-e^{-x}}{e^x+e^{-x}}$ | 输出(-1,1)，均值为0 |
| **ReLU** | $f(x) = \max(0, x)$ | 计算快，缓解梯度消失 |
| **Softmax** | $f(x_i) = \frac{e^{x_i}}{\sum e^{x_j}}$ | 多分类输出层 |

### 3. 网络结构

```
输入层 → 隐藏层1 → 隐藏层2 → ... → 输出层
```

- **前馈神经网络（FNN）**：信号单向传播
- **卷积神经网络（CNN）**：适合图像处理
- **循环神经网络（RNN）**：适合序列数据

### 4. 学习算法

- **前向传播**：计算输出
- **损失函数**：衡量预测与真实值的差距
  - 均方误差（MSE）：回归任务
  - 交叉熵损失：分类任务
- **反向传播（BP）**：计算梯度，更新权重

$$w \leftarrow w - \eta \cdot \frac{\partial L}{\partial w}$$

- $\eta$：学习率
- **梯度下降**：批量梯度下降、随机梯度下降（SGD）、小批量梯度下降

### 5. 深度学习

- **深度神经网络**：多层隐藏层
- 关键技术：Dropout、Batch Normalization、残差连接
- 代表模型：AlexNet、VGG、ResNet、Transformer

## 四、机器学习的典型应用

| 领域 | 应用 |
|------|------|
| **计算机视觉** | 图像分类、目标检测、人脸识别 |
| **自然语言处理** | 机器翻译、文本分类、情感分析 |
| **推荐系统** | 电商推荐、视频推荐、音乐推荐 |
| **金融** | 风控、量化交易、信用评估 |
| **医疗** | 疾病预测、医学影像诊断 |
| **自动驾驶** | 环境感知、路径规划 |
| **语音** | 语音识别、语音合成 |

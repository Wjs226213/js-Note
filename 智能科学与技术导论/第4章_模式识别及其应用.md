## 一、模式识别的基本原理与方法

### 1. 基本概念

- **模式（Pattern）**：对事物或现象的定量或结构描述
- **模式类（Class）**：具有某些共同特性的模式的集合
- **模式识别**：将模式分到各自所属的模式类中

### 2. 模式识别系统的基本流程

```
原始数据 → 预处理 → 特征提取 → 特征选择 → 分类决策 → 识别结果
```

| 步骤 | 说明 |
|------|------|
| **数据获取** | 通过传感器获取原始数据 |
| **预处理** | 去噪、归一化、增强等 |
| **特征提取** | 从原始数据中提取有区分性的特征 |
| **特征选择** | 选择最有效的特征子集 |
| **分类决策** | 使用分类器对特征进行分类 |

### 3. 模式识别方法分类

| 类别 | 方法 | 特点 |
|------|------|------|
| **监督学习** | 贝叶斯、SVM、神经网络 | 需要标注好的训练样本 |
| **无监督学习** | K-Means、层次聚类 | 不需要标注数据 |
| **半监督学习** | 标签传播 | 少量标注+大量未标注 |

## 二、贝叶斯决策理论

### 1. 贝叶斯公式

$$P(\omega_i | x) = \frac{P(x|\omega_i) \cdot P(\omega_i)}{P(x)}$$

- $P(\omega_i | x)$：**后验概率** — 观察到特征x后，样本属于$\omega_i$类的概率
- $P(x|\omega_i)$：**类条件概率（似然）** — $\omega_i$类中出现特征x的概率
- $P(\omega_i)$：**先验概率** — 样本属于$\omega_i$类的概率
- $P(x)$：**证据** — 特征x出现的概率

### 2. 贝叶斯决策规则

- **最小错误率决策**：选择后验概率最大的类

$$\text{决策}: x \in \omega_k \quad \text{if} \quad P(\omega_k|x) = \max_i P(\omega_i|x)$$

- 等价于：选择 $P(x|\omega_i) \cdot P(\omega_i)$ 最大的类

### 3. 最小风险决策

- 考虑错误分类带来的**损失**：

$$R(\alpha_i|x) = \sum_j \lambda(\alpha_i|\omega_j) \cdot P(\omega_j|x)$$

- $\lambda(\alpha_i|\omega_j)$：将$\omega_j$类样本决策为$\omega_i$类时的损失
- 选择**风险最小**的决策

### 4. 贝叶斯分类的特点

- **最优性**：在已知先验概率和类条件概率时，贝叶斯分类器是最优的
- **需要概率模型**：需要估计概率分布
- **适用条件**：特征相互独立或分布已知

## 三、判别函数分类器

### 1. 判别函数

- 不直接估计概率，而是直接设计**判别函数**来划分决策区域
- 对于C类问题，需要设计判别函数 $g_1(x), g_2(x), ..., g_C(x)$
- **决策规则**：$x \in \omega_k$ 如果 $g_k(x) = \max_i g_i(x)$

### 2. 线性判别函数

$$g(x) = w^T x + w_0$$

- $w$：权向量
- $w_0$：阈值（偏置）
- 决策边界是**超平面**

### 3. 感知器算法

- 用于**线性可分**问题
- 通过迭代更新权向量：

$$w(k+1) = w(k) + \eta \cdot x_k \quad (\text{误分类时})$$

- $\eta$：学习率
- 当数据线性可分时，感知器算法**必定收敛**

### 4. 支持向量机（SVM）

- 寻找**最大间隔**超平面
- 通过**核函数**处理非线性可分问题
- 常用核函数：线性核、多项式核、RBF（高斯核）、Sigmoid核

## 四、模糊模式识别

### 1. 基本概念

- **模糊集合**：隶属度取值在[0,1]之间的集合
- **隶属度函数**：描述元素属于模糊集合的程度

### 2. 模糊识别方法

- **最大隶属度原则**：样本归属于隶属度最大的类别
- **择近原则**：计算样本与各类模糊集的贴近度，选择最接近的类

### 3. 应用

- 手写字符识别
- 医学诊断
- 图像分割

## 五、聚类分析

### 1. 基本思想

- 将数据集划分为若干个**组（簇）**
- 目标：**组内相似度高，组间相似度低**
- 属于**无监督学习**

### 2. 典型算法

#### K-Means 算法

```
1. 随机选择 K 个初始聚类中心
2. 将每个样本分配到最近的聚类中心
3. 重新计算各簇的中心
4. 重复步骤2-3直到收敛
```

- **优点**：简单、高效
- **缺点**：需预先指定K值、对初始中心敏感、只能发现球形簇

#### 层次聚类

- **凝聚型（自底向上）**：每个样本初始为一个簇，逐步合并
- **分裂型（自顶向下）**：所有样本初始为一个簇，逐步分裂
- 输出：**树状图（Dendrogram）**

#### DBSCAN

- 基于**密度**的聚类
- 能发现**任意形状**的簇
- 能自动识别**噪声点**
- 核心概念：ε-邻域、核心点、边界点、噪声点

### 3. 聚类评估

- **轮廓系数**：衡量聚类质量
- **Calinski-Harabasz 指数**：组间方差/组内方差
